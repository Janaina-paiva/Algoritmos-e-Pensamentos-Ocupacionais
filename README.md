# Algoritmos-e-Pensamentos-Ocupacionais
Análise e Desenvolvimento de Sistemas


DESCRIÇÃO

A disciplina Algoritmo e Pensamento Computacional apresenta os conceitos básicos para resolver problemas de forma lógica e estruturada, desenvolvendo habilidades essenciais de programação e construção de algoritmos.

TECNOLOGIAS

Os estudos envolvem os fundamentos da programação com foco na linguagem C, abordando raciocínio lógico, características das linguagens e manipulação de dados.

OBJETIVO DO PROJETO

Aplicar, por meio da linguagem C, os conteúdos vistos no semestre, utilizando estruturas sequenciais, condicionais, de repetição, estruturas de dados e modularização.
